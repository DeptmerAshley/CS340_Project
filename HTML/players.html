PLAYERS PAGE

The players page allows the user to view all players in the league and add a new player to the league. Each player that is added to the league must be added to a team. The player information that is recorded when adding a player it the player name, email, birthday, and team they play for. The players page also contains the delete option. The user may select a player from a dropdown menu and delete that player from the database.
